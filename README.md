# Kortex AI - Site Officiel

Bienvenue sur le dépôt du site officiel **Kortex AI**, un portfolio, blog et plateforme de formation dédiée à l'intelligence artificielle.

---

## 🚀 Description

Ce site est développé avec **Next.js 15** (App Router), **TypeScript**, **Tailwind CSS** et intègre la bibliothèque **shadcn/ui** pour des composants UI modernes et responsives.  
Il présente :

- Mes projets en intelligence artificielle
- Mes formations téléchargeables
- Mes articles techniques et tutoriels
- Une page « À propos » avec ma présentation

---

## 📂 Structure du projet

```

src/
├── app/                  # Pages et layouts Next.js App Router
├── components/           # Composants React réutilisables
│   └── ui/               # Composants shadcn/ui
├── lib/                  # Fonctions utilitaires
├── public/               # Ressources statiques (images, favicon...)
├── styles/               # Fichiers CSS globaux

````

---

## 🛠️ Technologies utilisées

- [Next.js 15](https://nextjs.org)
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS v4](https://tailwindcss.com)
- [shadcn/ui](https://ui.shadcn.com)
- [Framer Motion](https://www.framer.com/motion/) pour les animations

## ⚙️ Installation

1. Cloner le dépôt :

```bash
git clone https://github.com/ton-utilisateur/kortexai.git
cd kortexai
````

2. Installer les dépendances :

```bash
npm install
```

3. Lancer le serveur de développement :

```bash
npm run dev
```

Le site sera accessible sur [http://localhost:3000](http://localhost:3000).

---

## 📦 Déploiement

* Le site est prêt à être déployé sur **Vercel** avec gestion du domaine personnalisé `kortexai.dev`.
* Configuration Tailwind CSS et shadcn/ui déjà en place.
* Backend minimal avec possibilité d’intégrer Supabase ou API personnalisées.
